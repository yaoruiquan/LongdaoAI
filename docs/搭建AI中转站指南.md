# 搭建 AI API 中转站指南

> 适用项目：龙道 AI / sub2api  
> 目标：把现成的开源中转程序部署到自己的服务器，卖 AI API 额度给用户

---

## 一、需要准备什么

| 东西 | 用途 | 大概花多少 | 哪里买 |
|---|---|---|---|
| **VPS 服务器** | 跑程序 | ¥75/月（本项目选用日本方案） | 国内云厂商日本节点 |
| **域名** | 给用户一个好看的地址 | ¥40–80/年 | 阿里云/腾讯云/Namecheap |

## 二、各项详细说明

### 服务器（VPS）

**香港 vs 日本对比（同为 2核4G，国内云厂商弹性套餐）：**

| 项目 | 香港 CN2 GIA | 日本 BGP（本项目选用） |
|---|---|---|
| 区域 | 香港 | 日本 |
| 系统 | Debian 12 x64 | Debian 12 x64 |
| CPU / 内存 | 2 核 / 4 GB | 2 核 / 4 GB |
| 硬盘 | 40 GB | 30 GB |
| **带宽** | 5 Mbps | **100 Mbps** |
| 流量 | 未标（5M 封顶） | 500 GB/月 |
| 延迟（到国内） | 30–50ms | 50–80ms |
| IP 干净度 | 部分 AI 官方封香港 | 更干净，官方基本不封 |
| **价格** | ¥111/月 | **¥75/月** |

**为什么选日本方案：**

- ✅ **更便宜**：¥75 vs ¥111，省 ¥36/月
- ✅ **带宽快 20 倍**：100Mbps vs 5Mbps，并发流式请求多了也不卡，余量足
- ✅ **IP 干净、留后路**：以后若想跳过中转站、自己拿官方 key 直连（成本更低、利润更高），日本 IP 不用换服务器；香港则可能被官方封，得重买重部署
- ✅ 500G 流量够用：纯文字 LLM 每次几 KB，能扛几百万次请求

**香港唯一的优势**是延迟低 20–30ms，但**用户体感几乎无差别**，不值得为此多花钱且带宽砍到 1/20。

> 别买 200Mbps 那档（¥175/月）——100Mbps 这阶段完全够，属于浪费。
> 若以后要转发图片/多模态导致流量吃紧，再升级不迟。

**为什么选国内大厂节点（而不是海外小厂）：**

- ✅ 支持**微信/支付宝**付款
- ✅ 能**开发票**（做生意/报销必需）
- ✅ 稳定性好、售后有保障

> 海外小厂（VMISS、GigsGigsCloud 等）虽然便宜，但多数不支持微信/支付宝、
> 也开不了发票，不适合正经做生意。

### 域名

随便一个就行，`.com` 最通用，`.net`/`.io` 也可以。

| 渠道 | 价格 | 说明 |
|---|---|---|
| 阿里云（万网） | ¥55–85/年 | 国内备案方便，支付宝付款 |
| 腾讯云 | ¥45–79/年 | 同上 |
| Namecheap | $10–12/年 | 海外购，无需备案 |
| Cloudflare | 成本价（约 $8–10/年） | 最便宜，自带 DNS 和 CDN |

> **注意**：服务器在香港/日本，不需要域名备案。如果服务器在大陆才需要。

---

### 上游 API（最重要的成本）

你自己不直接对接 OpenAI，而是买别的中转站的额度，再转卖给用户。

| 类型 | 特点 | 价格参考 |
|---|---|---|
| 国内 AI 中转站 | 免维护，按 token 计费，稳定 | 比官方便宜 20–50% |
| OpenAI 官方 | 最稳，需要境外信用卡 | 官方定价 |

常见国内中转站：CloseAI、OneAPI 聚合等，自己搜"AI API 中转 充值"找口碑好的。

> 这块是**变动成本**，用多少花多少，初期可以少充点测试。

## 三、简单搭建流程

```
第一步：买好 VPS + 域名
    ↓
第二步：域名解析到 VPS 的 IP
    （在域名控制台加一条 A 记录）
    ↓
第三步：SSH 登录服务器，跑部署脚本
    （项目已有现成的一键部署脚本）
    ↓
第四步：配置生产密钥（数据库密码、JWT等）
    ↓
第五步：脚本自动完成：
    - 拉取 Docker 镜像
    - 启动数据库、Redis、应用
    - Caddy 自动申请 HTTPS 证书
    ↓
第六步：访问你的域名，完成管理员初始化
    ↓
第七步：填入上游 API Key，创建测试用户
    ↓
上线完成，内测开始
```

整个过程熟悉了大约 **30–60 分钟**。

## 五、哪些不需要

❌ 不需要买住宅 IP  
❌ 不需要装 V2Ray / x-ui  
❌ 不需要在服务器上配科学上网  
❌ 不需要备案（服务器在境外）

上游用的是公开的 HTTPS API 地址，香港/日本服务器直接能连，零额外配置。

## 六、封闭内测 vs 公开收费

| | 封闭内测 | 公开收费 |
|---|---|---|
| 注册 | 关闭，管理员手动建账号 | 开放注册 |
| 充值 | 管理员手动加余额 | 需接入支付宝/微信支付 |
| 技术门槛 | **现在就能做** | 还需要开发支付模块 |
| 风险 | 低 | 涉及资金，需要完整对账系统 |

**建议先跑封闭内测，验证稳定性后再开放收费。**
